🚀 Shopify Label Manager - Quick Install

עברור 1: העתק את הקוד

קליק על הכפתור וקוד אוטומטית יעבור ללוח הגזירה שלך:

שלב 2: כנס לעיצוב שלך

  1. כנס ל-Shopify Admin
  2. Settings → Online Store → Themes
  3. לחץ "Edit code" בעיצוב הפעיל

שלב 3: מצא את theme.liquid

  1. בתפריט השמאלי, בחר layout
  2. לחץ על theme.liquid

שלב 4: הוסף את הקוד

  1. גרול לתחתית הקובץ
  2. חפש את </body>
  3. הדבק את הקוד לפני </body>

שלב 5: שמור

  1. לחץ Save בפינה הימנית העליונה
  2. חכה 30 שניות
  3. כנס לחנות שלך וראה את היישומון בפינה הימנית התחתונה! ✅
